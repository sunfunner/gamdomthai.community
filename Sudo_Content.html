คำสั่ง Sudo
Sudo = Super User Do
Group wheel เป็นค่า Default ของ CentOS ใช้สิทธิได้เหมือน Root
จะมีการให้ใส่ Password ยืนยัน

sudo dhclient -- สร้าง IP ต่อเน็ต
sudo yum install [Program] -- ลงโปรแกรม
sudo yum update -- อัพเดทโปรแกรม
sudo dhclient -- สร้าง IP เชื่อมต่อเน็ต

Program install
- tree
- blin-utils
- httpd
- net-tools
